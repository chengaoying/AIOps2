# 16 - 系统配置页面详细设计

**创建时间**: 2026-05-06
**状态**: 待实现

---

## 概述

系统配置页面提供系统级参数的配置管理，包括诊断规则、告警阈值、LLM设置等。

### 页面定位

- **入口**: Sidebar "系统配置 → 系统配置"
- **功能**: 系统参数配置、规则管理、通知渠道
- **用户**: 系统管理员

## Tab 2: 告警阈值

### 阈值配置表

| 指标 | 条件 | 默认值 | 说明 |
|------|------|--------|------|
| 作业失败数/小时 | > | 10 | 超过则告警 |
| 作业平均耗时 | > | 300s | 超过则告警 |
| Executor OOM率 | > | 5% | OOM作业占比 |

## API 接口

### 获取诊断规则

```go
// GET /api/v1/settings/system/rules
type GetRulesResponse struct {
    Rules []Rule `json:"rules"`
}

type Rule struct {
    ID          int64  `json:"id"`
    Name        string `json:"name"`
    Type        string `json:"type"`        // rule/ai
    Enabled     bool   `json:"enabled"`
    Priority    string `json:"priority"`    // high/medium/low
    Conditions  []Condition `json:"conditions"`
    RootCause   string `json:"root_cause"`
    Suggestions []string `json:"suggestions"`
    MatchCount  int    `json:"match_count"`
}

type Condition struct {
    Metric   string `json:"metric"`
    Operator string `json:"operator"`    // >, <, =, !=
    Value    string `json:"value"`
}
```

### 更新规则

```go
// PUT /api/v1/settings/system/rules/{rule_id}
type UpdateRuleRequest struct {
    Enabled     *bool        `json:"enabled,omitempty"`
    Priority    *string      `json:"priority,omitempty"`
    Conditions  []Condition  `json:"conditions,omitempty"`
    RootCause   *string      `json:"root_cause,omitempty"`
    Suggestions []string     `json:"suggestions,omitempty"`
}
```

### 获取告警阈值

```go
// GET /api/v1/settings/system/thresholds
type GetThresholdsResponse struct {
    Thresholds []Threshold `json:"thresholds"`
}

type Threshold struct {
    ID          int64   `json:"id"`
    Metric      string  `json:"metric"`
    Operator    string  `json:"operator"`
    Value       float64 `json:"value"`
    Level       string  `json:"level"`       // low/medium/high
    Description string  `json:"description"`
}
```

### 更新阈值

```go
// PUT /api/v1/settings/system/thresholds/{threshold_id}
type UpdateThresholdRequest struct {
    Operator string  `json:"operator"`
    Value    float64 `json:"value"`
    Level    string  `json:"level"`
}
```

### 获取LLM配置

```go
// GET /api/v1/settings/system/llm
type GetLLMConfigResponse struct {
    Provider   string  `json:"provider"`
    Model      string  `json:"model"`
    Temperature float64 `json:"temperature"`
    MaxTokens  int     `json:"max_tokens"`
    Timeout    int     `json:"timeout"`
}
```

### 更新LLM配置

```go
// PUT /api/v1/settings/system/llm
type UpdateLLMConfigRequest struct {
    Provider    string  `json:"provider"`
    APIKey      string  `json:"api_key,omitempty"`
    Model       string  `json:"model"`
    Temperature float64 `json:"temperature"`
    MaxTokens   int     `json:"max_tokens"`
    Timeout     int     `json:"timeout"`
}
```
