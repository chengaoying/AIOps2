# 11 - 元仓页面详细设计

**创建时间**: 2026-05-06
**状态**: 待实现

---

## 概述

元仓（Metastore）页面展示作业元数据信息，根据组件类型展示不同的监控视图：YARN 展示计算资源和队列情况，Hive/Spark/Flink 展示作业详情。

### 页面定位

- **入口**: Sidebar "元仓"
- **功能**: 组件类型切换、资源/作业视图、趋势分析
- **用户**: 运维工程师查看资源使用和作业链路
