# 07 - Dashboard 详细设计

**创建时间**: 2026-05-06
**状态**: 待实现

---

## 概述

Dashboard 是用户进入系统的第一个页面，包含诊断表单和诊断结果展示。

### 设计方向（从 design-shotgun 生成 3 个变体）

| 变体 | 风格 | 特点 |
|------|------|------|
| A1 | 专业控制室风格 | 高信息密度，适合运维 |
| A2 | 温暖向导风格 | 友好，易上手 |
| A3 | 极简科技风格 | 最小化，更专注 |

---

### 组件规范

#### 诊断表单卡片

- 白色背景 (#FFFFFF)
- 8px 圆角
- 边框 1px solid #E5E5E5
- 内边距 24px

#### 平台选择器

- 胶囊按钮组
- 默认态：灰色背景
- 选中态：平台颜色背景，白色文字
- Hover：平台颜色浅色背景

#### 输入框

- 高度 40px
- 边框 #E5E5E5
- Focus：蓝色边框 (#2563EB)
- Placeholder 灰色 (#9CA3AF)

#### 主按钮

- 背景 #2563EB
- 文字白色
- 高度 40px
- Hover：#1D4ED8
- Loading：显示 spinner

#### 诊断结果卡片

- 左边框 4px 平台颜色
- 标题区：JobID + 平台标签 + 置信度标签
- 根因区：🔍 图标 + 文字
- 建议区：📋 图标 + 编号列表

### 交互行为

| 交互 | 行为 |
|------|------|
| 选择平台 | 高亮选中项，表单验证更新 |
| 输入作业ID | 实时验证格式 |
| 点击诊断 | 按钮 loading，显示诊断进度 |
| 诊断完成 | 结果卡片 fade-in，滚动到结果 |

## 通用组件

### PlatformSelector

```tsx
interface PlatformSelectorProps {
    value: Platform | null;
    onChange: (platform: Platform) => void;
    options?: Platform[]; // 默认全部
}
```

### DiagnosisForm

```tsx
interface DiagnosisFormProps {
    onSubmit: (platform: Platform, jobId: string) => void;
    loading?: boolean;
    error?: string;
}
```

### DiagnosisResult

```tsx
interface DiagnosisResultProps {
    jobId: string;
    platform: Platform;
    confidence: number;
    rootCause: string;
    suggestions: Suggestion[];
    onSuggestionClick?: (suggestion: Suggestion) => void;
}
```

### ConfidenceBadge

```tsx
const getConfidenceColor = (confidence: number) => {
    if (confidence >= 0.9) return '#10B981'; // 绿色
    if (confidence >= 0.7) return '#F59E0B'; // 黄色
    return '#EF4444'; // 红色
};

interface ConfidenceBadgeProps {
    value: number; // 0-1
}
```

### RiskBadge

```tsx
interface RiskBadgeProps {
    level: 'low' | 'medium' | 'high';
}
```

## 响应式设计

| 断点 | 侧边栏 | 布局 |
|------|--------|------|
| > 1024px | 200px | 完整布局 |
| 768-1024px | 64px | 图标+文字 |
| < 768px | 0 | 汉堡菜单，侧滑 |

---

## 设计决策

| 决策 | 选择 | 理由 |
|------|------|------|
| 推荐方案 | A1 专业控制室 | 高信息密度，适合3am运维场景 |
| 侧边栏宽度 | 200px | 文字清晰，便于快速识别 |
| 主色调 | #2563EB | 专业、信任感 |
| 结果卡片 | 内嵌页面 | 减少跳转，保持上下文 |
